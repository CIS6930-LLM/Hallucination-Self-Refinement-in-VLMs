# Configuration Files

Store experiment and model configuration files here. Suggested organization:

- `baseline/`: Reproduction configs for BLIP-2, LLaVA, MiniGPT-4.
- `self_refinement/`: Variants toggling rationale, contrastive, and RLAIF modules.
- `evaluation/`: CLI settings for POPE, VQA-Hallu, and synthetic benchmarks.

Prefer human-readable formats (YAML/TOML) and document key hyperparameters inline.
